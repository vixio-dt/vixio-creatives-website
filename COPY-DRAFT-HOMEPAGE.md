## Copy Notes

**Key changes from current site:**
- "Premium physical products and playable experiences" → "Creative label for entertainment IPs"
- Product-specific language (plush, mystery kits, DIY builds, puzzle boxes) → removed. Medium is not locked.
- "Physical-first, digitally enhanced" → "The through-line is taste, not genre"
- "GBA manufacturing" → "Dongguan production network" (more specific, same meaning)
- Services list (5 product types) → Label process (find/develop/produce/release)
- Problem framing: "everything is a screen" → "worlds built with care, products made without it"
- "First Drop List" → "First Release List" (vocabulary alignment)
- HowItWorks steps: Concept/Engineer/Ship → Partner/Create/Release

**What stays:**
- Dual-path structure (fans + creators)
- Direct, compressed voice
- Pre-launch honesty ("producing our first collaboration")
- Dark/cream section alternation
- Contact form structure (email for fans, full form for creators)

**What the copy does NOT do:**
- Name a specific product format (intentional — medium emerges from collaboration)
- Claim anything not yet true (no shipped product, no portfolio, no testimonials)
- Use deficit framing ("we're new but...")
- Use banned vocabulary (innovative, disruptive, immersive as lead, etc.)
